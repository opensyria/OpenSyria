ℹ️ About RandomX Mining

OpenSY uses RandomX, a CPU-friendly proof-of-work algorithm. GPU and ASIC mining is not efficient. Expected hashrates:

  • Intel i5/AMD Ryzen 5: ~500-1500 H/s
  • Intel i7/AMD Ryzen 7: ~1500-3000 H/s
  • Intel i9/AMD Ryzen 9: ~3000-5000 H/s
  • Server CPUs (Xeon/EPYC): ~5000-15000 H/s
